You can push through unlimited density of enemies without slowing down.4. Developer Implementation NotesArmor Shred Math (Orange/Cyan)Orange interactions often refer to "Reducing Armor." Since enemies may not have complex stats, implement this as a Damage Multiplier Tag.JavaScript// Pseudo-code for Damage Calculation
function calculateDamage(enemy, baseDamage) {
  let multiplier = 1.0;
  if (enemy.hasEffect("CORRODED")) multiplier += 0.3; // Cyan+Orange
  if (enemy.hasEffect("ENTROPY")) multiplier += 0.5; // Orange+Cyan (Melee)
  
  return baseDamage * multiplier;
}
